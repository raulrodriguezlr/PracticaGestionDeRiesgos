# Motor de Stress Testing en Python

**Practica B2-2 - Modulo de Gestion de Riesgos - MIAX**

## Descripcion

Motor de stress testing que detecta regimenes de mercado con HMM, modela dependencias con copulas, simula escenarios con Monte Carlo y cuantifica riesgo extremo (VaR 99%, CVaR).

## Uso en Google Colab

1. Subir `main.ipynb` a Google Colab
2. Ejecutar todas las celdas (Ctrl+F9)
3. La primera celda instala las librerias necesarias

## Archivos

| Archivo | Descripcion |
|---------|-------------|
| `main.ipynb` | Notebook principal (entregable) |
| `dev.ipynb` | Notebook de desarrollo |
| `prompt.txt` | Enunciado de la practica |
| `requirements.txt` | Dependencias Python |
